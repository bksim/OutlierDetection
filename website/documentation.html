This page provides documentation for various functions which facilitate the pipeline of time sereies feature extraction to outlier detection.

We also create a class LightCurve which serves as a wrapper class for a light curve, holding all its features and feature values, and providing wrappers to functions to calculate these features for a given light curve.

Finally, we provide a flexible function for computing outliers via any outlier detection function in our extensive library of outlier detection functions, described on the methods page. This function is as follows:
